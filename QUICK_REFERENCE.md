# 📖 OpenAPI Quick Reference\n\n**Быстрый справочник для разработчиков**\n\n---\n\n## 🎯 B2B: Seller Cabinet\n\n### Products\n```\nGET    /api/v1/products              List products (filter, pagination)\nPOST   /api/v1/products              Create product\nGET    /api/v1/products/{id}         Get product with SKUs\nPUT    /api/v1/products/{id}         Update product\n```\n\n### SKUs\n```\nPOST   /api/v1/skus                  Create SKU\nPUT    /api/v1/skus                  Update SKU\n```\n\n### Invoices\n```\nPOST   /api/v1/invoices              Create invoice\nPOST   /api/v1/invoices/accept       Accept invoice (warehouse)\n```\n\n**Security:** BearerAuth (JWT)  \n**OpenAPI:** `b2b/openapi.yaml`\n\n---\n\n## 🛒 B2C: Cart + Favorites + Home\n\n### Favorites (❤️ Wishlist)\n```\nGET    /api/v1/favorites             List favorites (pagination)\nPOST   /api/v1/favorites/{product_id}      Add to favorites\nDELETE /api/v1/favorites/{product_id}      Remove from favorites\n```\n\n### Subscriptions (🔔 Notifications)\n```\nPOST   /api/v1/favorites/{product_id}/subscribe\n       → notify_on: [IN_STOCK, PRICE_DOWN]\n```\n\n### Cart (🛍️ Shopping)\n```\nGET    /api/v1/cart                  View cart (actual prices)\nDELETE /api/v1/cart                  Clear cart\n\nPOST   /api/v1/cart/items            Add item\nGET    /api/v1/cart/items/{item_id}  Get cart item\nPUT    /api/v1/cart/items/{item_id}  Update quantity\nDELETE /api/v1/cart/items/{item_id}  Remove item\n```\n\n### Validation (✅ Check before checkout)\n```\nGET    /api/v1/cart/validate         Validate cart\n       ← is_valid, can_checkout, issues[]\n```\n\n### Home Page Banners (🎨 Marketing)\n```\nGET    /api/v1/home/banners          Get active banners\nPOST   /api/v1/banner-events         Send banner analytics\n       → event: impression | click\n```\n\n### Collections (📚 Curated lists)\n```\nGET    /api/v1/collections/{id}/products  Products in collection\nGET    /api/v1/main/collections           List all collections\n```\n\n### Recommendations (💡 Smart suggestions)\n```\nGET    /api/v1/cart/also_bought      \"Customers also bought\" items\n```\n\n**Security:** BearerAuth (JWT) + X-User-Id / X-Session-Id  \n**OpenAPI:** `b2c/cart/openapi.yaml`\n\n---\n\n## 📚 B2C: Catalog + Search\n\n### Products (🔍 Search & Filter)\n```\nGET    /api/v1/products              Search with filters\n       ?search=...&category_id=...&filters=...&sort=...\n       → sort: rating|popularity|price_asc|price_desc|date_desc\n\nGET    /api/v1/products/{id}         Full product details\nGET    /api/v1/products/{id}/similar Similar products\n```\n\n### SKUs (📦 Product variants)\n```\nGET    /api/v1/products/{id}/skus           List SKUs (brief)\nGET    /api/v1/products/{id}/skus/{sku_id} SKU details\n```\n\n### Categories (📂 Navigation)\n```\nGET    /api/v1/categories            Tree of all categories\nGET    /api/v1/categories/{id}       Category details + SEO\nGET    /api/v1/categories/{id}/filters  Available filters for category\n```\n\n### Facets (🏷️ Dynamic filters)\n```\nGET    /api/v1/catalog/facets        Facets with item counts\n       ?category_id=...&filters=...\n       ← [{ name, slug, type, values: [{value, count, is_selected}] }]\n```\n\n### Breadcrumbs (🗺️ Navigation path)\n```\nGET    /api/v1/breadcrumbs           Navigation chain\n       ?category_id=... OR ?product_id=...\n```\n\n**Security:** Public (some endpoints require BearerAuth)  \n**OpenAPI:** `b2c/catalog/openapi.yaml`\n\n---\n\n## 📦 B2C: Orders (📋 Checkout & History)\n\n### Orders (✨ Full lifecycle)\n```\nPOST   /api/v1/orders                Create order\n       { items[], total, delivery_address, payment_method }\n\nGET    /api/v1/orders                Order history (pagination)\nGET    /api/v1/orders/{order_id}     Order details\n```\n\n### Order Management\n```\nPOST   /api/v1/orders/{order_id}/cancel     Cancel order\nPATCH  /api/v1/orders/{order_id}/status     Update status (admin)\n       statuses: PENDING → PAID → ASSEMBLING → SHIPPED → DELIVERED\n```\n\n**Payment Methods:**\n- `CARD_ONLINE` — Credit/Debit card\n- `CASH_ON_DELIVERY` — Pay on delivery\n\n**Order Statuses:**\n```\nPENDING    → New order, payment pending\nPAID       → Payment received\nASSEMBLING → Order being packed\nSHIPPED    → On the way\nDELIVERED  → Delivered to customer\nCANCELED   → Order cancelled\n```\n\n**Security:** BearerAuth (JWT)  \n**OpenAPI:** `b2c/orders/openapi.yaml`\n\n---\n\n## 👮 Moderation: Product Review Queue\n\n### Queue Management (📋 Moderation queue)\n```\nPOST   /api/v1/product-moderation/get-next    Next card to review\nPOST   /api/v1/product-moderation/enqueue     Manually enqueue product\n```\n\n### Decisions (✅ Accept/Reject)\n```\nPOST   /api/v1/products/{id}/approve          Approve product\nPOST   /api/v1/products/{id}/decline          Decline (block) product\n       { reason_code, comment?, fields?[] }\n```\n\n### Reference (📚 Reason codes)\n```\nGET    /api/v1/product-blocking-reasons       List blocking reasons\n       ← [{ code, title, description }]\n```\n\n**Product Status Flow:**\n```\nCREATED         → Seller just created\nON_MODERATION   → In review queue\nMODERATED       → ✅ Approved, visible in catalog\nBLOCKED         → ❌ Declined, not visible\n```\n\n**Security:** Internal API (requires auth)  \n**OpenAPI:** `moderation/openapi.yaml`\n\n---\n\n## 🔐 Security Headers\n\n### BearerAuth (JWT)\n```\nAuthorization: Bearer <jwt_token>\n```\n\n### Cart Headers\n```\nX-User-Id: <user_uuid>        # For authenticated users\nX-Session-Id: <session_uuid>  # For guest users\n# Either X-User-Id OR X-Session-Id required\n```\n\n---\n\n## 📊 Common Response Codes\n\n```\n200 OK                   Successful request\n201 Created              Resource created\n204 No Content           Deleted successfully\n\n400 Bad Request          Invalid parameters\n401 Unauthorized         Missing/invalid auth\n403 Forbidden            Access denied\n404 Not Found            Resource not found\n409 Conflict             State conflict (e.g., already added)\n410 Gone                 Product unavailable/deleted\n422 Unprocessable        Validation error\n\n500 Internal Server      Server error\n503 Service Unavailable  Upstream service down\n```\n\n---\n\n## 🚫 Common Error Codes\n\n```\nNOT_FOUND                    Resource not found\nUNAUTHORIZED                 Auth required\nACCESS_DENIED                No permission\nINVALID_REQUEST              Bad parameters\nINVALID_FORMAT               Wrong data format\nOUT_OF_STOCK                 Product unavailable\nINSUFFICIENT_STOCK           Not enough items\nPRODUCT_BLOCKED              Product down\nCARE_VALIDATION_ERROR        Unknown issue\n```\n\n---\n\n## 💾 Data Models: Quick Overview\n\n### Product\n```yaml\nid: uuid\ntitle: string\ndescription: string\nstatus: CREATED | ON_MODERATION | MODERATED | BLOCKED\ncategory: { id, name }\nimages: [{ url, ordering }]\ncharacteristics: [{ name, value }]\nskus: [SKU]\n```\n\n### SKU\n```yaml\nid: uuid\nname: string              # \"iPhone 15 Pro 256GB Black\"\nprice: integer            # в копейках (копейки)\nactive_quantity: integer  # на складе\ncharacteristics: [{ name, value }]\nimages: [{ url, order }]\n```\n\n### CartItem\n```yaml\nitem_id: uuid\nsku_id: uuid\nproduct_id: uuid\nquantity: integer\nunit_price: { amount, currency }\nline_total: { amount, currency }\navailable: boolean\n```\n\n### Order\n```yaml\nid: uuid\nuser_id: uuid\nstatus: OrderStatus\nitems: [OrderItem]    # {product_id, sku_id, quantity, prices}\ntotal: { amount: integer, currency: string }  # в копейках\npayment_method: CARD_ONLINE | CASH_ON_DELIVERY\ndelivery_address: DeliveryAddress\ncreated_at: ISO8601\nupdated_at: ISO8601\n```\n\n### CartValidationIssue\n```yaml\ncart_item_id: string\nsku_id: integer\nissue_type: BLOCKED | DELETED | OUT_OF_STOCK | INSUFFICIENT_STOCK | ON_MODERATION | VALIDATION_ERROR\nseverity: critical | warning\nmessage: string\ndetails: object\n```\n\n---\n\n## 🎯 Usage Examples\n\n### 1.
