# NeoMarket B2B Cabinet Frontend

Современный и красивый B2B кабинет для платформы NeoMarket, вдохновлённый дизайном Wildberries и Ozone.

## 🎨 Цветовая гамма

- **Primary (Teal):** `#0f766e` - основной цвет
- **Accent (Orange):** `#f97316` - акцент
- **Background:** `#f7f4ee` - светлый фон
- **Text:** `#1b2230` - тёмный текст

## 📦 Структура проекта

```
src/
├── App.jsx              # Главный компонент приложения
├── App.css              # Стили приложения
├── index.css            # Глобальные стили
├── main.jsx             # Точка входа
├── components/
│   ├── Header.jsx       # Шапка с поиском и профилем
│   ├── Header.css
│   ├── Sidebar.jsx      # Боковое меню навигации
│   └── Sidebar.css
└── pages/
    ├── Dashboard.jsx    # Главная панель управления
    ├── Dashboard.css
    ├── Catalog.jsx      # Каталог поставщиков
    ├── Catalog.css
    ├── Orders.jsx       # История заказов
    ├── Orders.css
    ├── Cart.jsx         # Корзина
    ├── Cart.css
    ├── Profile.jsx      # Профиль компании
    └── Profile.css
```

## 🚀 Установка и запуск

### Установка зависимостей
```bash
npm install
# или в корневой директории frontend:
npm run install-all
```

### Разработка
```bash
npm run dev
# Приложение откроется на http://localhost:5176
```

### Сборка
```bash
npm run build
```

## ✨ Особенности

### 1. **Dashboard** 📊
- Статистика выручки, заказов, рейтинга
- Рекомендации по улучшению продаж
- Последние заказы в реальном времени

### 2. **Каталог** 📦
- Поиск и фильтрация товаров от поставщиков
- Категории и сортировка
- Рейтинг поставщиков
- Быстрое добавление в корзину

### 3. **Корзина** 🛒
- Управление количеством товаров
- Расчёт с учётом скидок и доставки
- Несколько методов оплаты
- Рекомендуемые товары

### 4. **Заказы** 📋
- Фильтрация по статусу
- История всех закупок
- Скачивание счётов
- Статистика по расходам

### 5. **Профиль** 👤
- Редактирование информации о компании
- Статистика рейтинга и отзывов
- Параметры аккаунта
- Управление настройками

## 🎨 Design System

### Размеры шрифтов
- H1: 32px (заголовки страниц)
- H2: 20px (заголовки секций)
- H3: 16px (заголовки карточек)
- Body: 14px (основной текст)
- Small: 12px (вспомогательный текст)
- Tiny: 11px (labels)

### Spacing
- 4px, 8px, 12px, 16px, 20px, 24px, 32px

### Border Radius
- 4px (маленькие элементы)
- 6px (кнопки, поля)
- 8px (карточки)
- 12px (основные карточки)

## 🔌 API Integration

Приложение готово к интеграции с API:
- `/api/v1/b2b/dashboard` - статистика
- `/api/v1/b2b/products` - каталог товаров
- `/api/v1/b2b/orders` - история заказов
- `/api/v1/b2b/profile` - профиль компании

## 🎬 Анимации

- `slideUp` - появление элементов
- `slideDown` - выпадающие меню
- `fadeIn` - плавное появление
- `pulse` - пульсирующие элементы
