🚀 Quick Start

Each app is a standalone Vite + React application. Start developing:

cd frontend/apps/catalog
npm install
npm run dev

# In another terminal, start other apps:
cd frontend/apps/cart && npm run dev
cd frontend/apps/orders && npm run dev
cd frontend/apps/b2b-cabinet && npm run dev
cd frontend/apps/moderation && npm run dev

ℹ️ All apps make requests to http://localhost:8888/api/v1 (API Gateway).
Each service responds at: /api/v1/{service-name}/{endpoint}
